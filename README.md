# BodyPartBuddy - Interactive AI Mirror for Kids

BodyPartBuddy is an educational "Magic Mirror" application designed for children. It uses computer vision (MediaPipe & OpenCV) to detect body parts in real-time and label them with fun, colorful visuals.

## Features

*   **Real-time Detection:** Identifies Head, Arms, Hands, Legs, and Feet.
*   **Magic Mirror Effect:** The display acts like a mirror for intuitive interaction.
*   **Kid-Friendly UI:** Large text, bright neon colors, and simple shapes.
*   **Privacy Focused:** Runs entirely locally on your machine.

## Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/CardinalisUnum/EduComVi.git
    ```
2.  Install dependencies:
    ```bash
    pip install -r requirements.txt
    ```

## Usage

Run the main script:
```bash
python main.py
```
Press **Esc** to exit the application.

## Technologies

*   Python 3.x
*   OpenCV
*   MediaPipe
*   NumPy
